Specify the format for dates in the game.

[h1]Detailed Description[/h1]
The following date formatting items can be specified:[list]
[*]Date part order can be:  DMY, MDY, or YMD.  Default is DMY.
[*]Date part separator can be:  slash (/), period (.), dash (-), or space.  Default is slash.
[*]Month and day can have or not have a leading zero for values less than 10.  Default is with leading zeroes.[/list]

Game dates will often be year 2100 or later, therefore the year is always formatted with 4 digits.

Click Reset To Default to reset all settings to their default values.

The date format can be changed while a game is in progress.  The new date format will be applied immediately.

The following dates are formatted:[list]
[*]Main game date
[*]ChirpX Launch Site - past launch dates
[*]Festival Area - concert dates
[*]Football Stadium - next and past match dates
[*]Varsity Sports Arenas - next and past match dates[/list]

This mod does not format the date/time displayed when loading or saving:  games, maps, themes, assets, and scenarios.

[h1]Compatibility[/h1]
This mod does not alter game save files in any way and can be safely disabled or unsubscribed at any time.

This mod does not alter the game mechanics or the game date in any way.  This mod only changes how game dates are displayed.

This mod has been tested only on Windows but should work on Mac and Linux.

This mod is compatible with all DLC packages thru Sunset Harbor and all Content Creator Packs (CCP) thru Train Stations/Bridges & Piers.  This mod does not require any DLC or CCP to work properly.

This mod is compatible with the following mods.  The main game date will be formatted when any of these mods is enabled.[list]
[*][url=https://steamcommunity.com/sharedfiles/filedetails/?id=781767563]Extended InfoPanel[/url]
[*][url=https://steamcommunity.com/sharedfiles/filedetails/?id=2498761388]Extended InfoPanel 2[/url]
[*][url=https://steamcommunity.com/sharedfiles/filedetails/?id=2274354659L]Extended InfoPanel 21:9[/url][/list]

This mod is a replacement for and therefore is not compatible with the [url=https://steamcommunity.com/sharedfiles/filedetails/?id=565071445]Date Reformatter[/url] mod.
